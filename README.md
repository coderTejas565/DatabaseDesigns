# Database Designs

A curated collection of real-world database design projects built as part of a structured **5-day challenge** from the Chai Code.

This repository focuses on developing a strong foundation in **data modeling, relationships, and backend system design** through practical, hands-on implementation.

---

## Purpose

* Design real-world database systems from scratch
* Understand entity relationships and schema structure
* Build backend engineering mindset
* Practice scalable and maintainable data models
